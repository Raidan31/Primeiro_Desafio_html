Estrutura do HTML

(voltar)

A estrutura básica de um documento HTML é composta por quatro elementos principais: html, head, title, body. Aqui está uma explicação breve de cada um:

HTML: É o elemento raiz que engloba todo o conteúdo da página.

HEAD: Contém informações sobre o documento, como o título da página, que é exibido na aba do navegador, e meta-informações, como a descrição da página.

TITLE: Especifica o título da página, que é exibido na aba do navegador.

BODY: Contém o conteúdo da página, incluindo texto, imagens, links e outros elementos HTML.

META: Fornece informações adicionais sobre a página, como a codificação de caracteres, a descrição e as palavras-chave para os mecanismos de busca.

Hiperlinks no html

(voltar)

A tag em HTML é utilizada para criar hiperlinks, permitindo que o usuário navegue de uma página para outra ou abra um documento ou recurso externo. É a principal forma de permitir a navegação na web. A sintaxe básica é a seguinte:

a href="URL">Texto do link

Onde href é um atributo que especifica o destino do link e Texto do link é o texto que será exibido para o usuário clicar. Além de href, a tag a também suporta outros atributos, como target, download e rel, entre outros.

Listas

 (voltar)

Em HTML, existem dois tipos principais de listas: listas não ordenadas e listas ordenadas.

Listas não ordenadas: São representadas pelo elemento ul e são usadas para criar uma lista de itens não numerados. Cada item da lista é representado pelo elemento li. Aqui está um exemplo:

ul>

li Item 1 /li

li Item 2 /li

li Item 3 /li

/ul>

Listas ordenadas: São representadas pelo elemento ol e são usadas para criar uma lista de itens numerados. Cada item da lista é representado pelo elemento li . Aqui está um exemplo:

ol

li Item 1 li

li Item 2 li

li Item 3 li

ol
